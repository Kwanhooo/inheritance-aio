## 🦾 源码

### 1. 安装依赖

```npm
npm install
```

### 2. 启动服务

```npm
npm run dev
```

### 3. 开发

```npm
# 注册 Husky Git 钩子
npm run prepare
```

```npm
# 格式化代码
npm run lint
npm run lintfix
```

```shell
# 提交代码

# 1.添加暂存区
git add .

# 2.提交代码
  # `type`的允许值详见 commitlint.config.js
git commit -m <type>[optional scope]: <description>

# 3.推送
git push
```
